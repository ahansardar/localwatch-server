# LocalWatch Server Privacy Notice

Effective: July 4, 2026

LocalWatch Server does not collect, upload, sell, or share personal data. The app does not require an account, analytics service, advertising service, or internet connection.

The host chooses a folder through Android's system document picker. The app receives read access only to that selected folder and uses it to list and stream supported video files. Media remains on the host device.

When the local server is running, devices that can reach the host over the local Wi-Fi network or hotspot may request the shared library. The host app temporarily displays recent client IP addresses in memory. These addresses are not uploaded or retained after the server stops or the app process ends.

LocalWatch uses unencrypted HTTP on the local network. The optional PIN limits casual access but is not a substitute for an encrypted, trusted network. The host should share only media they have the right to share and should enable PIN protection on shared networks.

Questions can be sent to [ahansardarvis@gmail.com](mailto:ahansardarvis@gmail.com), or opened publicly in the [LocalWatch issue tracker](https://github.com/ahansardar/localwatch-server/issues).
