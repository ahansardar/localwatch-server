# Changelog

All notable changes to LocalWatch are documented here. The project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-05

### Added

- Native Jetpack Compose host app with server-off and server-on dashboards.
- Folder selection through Android's Storage Access Framework.
- Foreground local HTTP server with QR, copy, share, and notification controls.
- Responsive offline web library with search, sort, grid/list views, PIN access, and player states.
- Byte-range video streaming and optional downloads.
- Settings, library, help, about, recent clients, rescanning, and theming.
- Daily GitHub release checks with Markdown notes, download progress, verification, and installer handoff.
- LocalWatch visual identity across the Android launcher, app, and browser UI.
