# Security Policy

## Supported versions

Security fixes are applied to the latest published LocalWatch release.

## Reporting a vulnerability

Please do not publish exploitable details in a public issue. Email **ahansardarvis@gmail.com** with:

- the affected version;
- the conditions required to reproduce the issue;
- the expected impact; and
- a minimal proof of concept, if available.

You should receive an acknowledgement within seven days. Once a fix is available, disclosure can be coordinated around the release.

## Scope notes

LocalWatch is intended for trusted local Wi-Fi or hotspot networks. Its optional PIN is an access control, not encryption, and the local HTTP stream is not protected against an attacker already able to observe that network.
