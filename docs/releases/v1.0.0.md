# LocalWatch v1.0.0 — your phone is now the server

This is the first public release of LocalWatch: a private, offline-first Android video server for the people already in the room.

## What you can do

- Choose a video folder using Android's secure system picker.
- Start a local server and share it by QR code, URL copy, or Android's share sheet.
- Browse a polished responsive library from phones, tablets, laptops, and TVs.
- Search, sort, switch between grid and list layouts, and stream with proper seeking.
- Protect the library with an optional PIN and decide whether viewers may download files.
- Rescan the folder, inspect recent clients, change the port, and keep the host awake.

## Built for the local network

The viewer experience is completely self-contained—no CDN, remote font, account, analytics service, or cloud media upload. Your phone serves the UI and videos directly over the current Wi-Fi network or hotspot.

## Updates are ready

LocalWatch checks this repository's latest release once a day. When a newer version appears, it shows these Markdown release notes, downloads the APK with visible progress, verifies its SHA-256 digest when GitHub provides one, and opens Android's installer.

## Compatibility

- Android 8.0 or newer is required on the host.
- MP4/H.264 offers the widest browser support.
- Other recognized containers can be downloaded or opened in VLC if the browser cannot decode them.
- Every viewer must be able to reach the host on the same Wi-Fi network or hotspot.
